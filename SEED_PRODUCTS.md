# Product Seeding Guide

рЃћрЃА рЃАрЃЎрЃарЃўрЃърЃбрЃў рЃљрЃЋрЃбрЃЮрЃЏрЃљрЃбрЃБрЃарЃљрЃЊ рЃерЃћрЃЦрЃЏрЃюрЃўрЃА рЃарЃљрЃЏрЃЊрЃћрЃюрЃўрЃЏрЃћ рЃАрЃљрЃбрЃћрЃАрЃбрЃЮ рЃърЃарЃЮрЃЊрЃБрЃЦрЃбрЃА Firestore рЃЉрЃљрЃќрЃљрЃерЃў.

## рЃЏрЃЮрЃЌрЃ«рЃЮрЃЋрЃюрЃћрЃЉрЃў

рЃАрЃљрЃюрЃљрЃЏ рЃърЃарЃЮрЃЊрЃБрЃЦрЃбрЃћрЃЉрЃўрЃА seeding-рЃА рЃњрЃљрЃБрЃерЃЋрЃћрЃЉрЃЌ, рЃЊрЃљрЃарЃгрЃЏрЃБрЃюрЃЊрЃўрЃЌ рЃарЃЮрЃЏ:

1. РюЁ **рЃўрЃБрЃќрЃћрЃарЃћрЃЉрЃў рЃБрЃЎрЃЋрЃћ рЃерЃћрЃЦрЃЏрЃюрЃўрЃџрЃўрЃљ** - рЃњрЃљрЃБрЃерЃЋрЃўрЃЌ `npm run seed:users` рЃърЃўрЃарЃЋрЃћрЃџ рЃарЃўрЃњрЃерЃў
2. РюЁ **Firebase credentials рЃЎрЃЮрЃюрЃцрЃўрЃњрЃБрЃарЃўрЃарЃћрЃЉрЃБрЃџрЃўрЃљ** - `.env` рЃцрЃљрЃўрЃџрЃерЃў
3. РюЁ **Firestore API рЃЕрЃљрЃарЃЌрЃБрЃџрЃўрЃљ** - Google Cloud Console-рЃерЃў

## рЃњрЃљрЃерЃЋрЃћрЃЉрЃљ

```bash
cd backend
npm run seed:products
```

## рЃарЃљрЃА рЃљрЃЎрЃћрЃЌрЃћрЃЉрЃА рЃАрЃЎрЃарЃўрЃърЃбрЃў

1. **рЃўрЃърЃЮрЃЋрЃюрЃўрЃА seller рЃўрЃБрЃќрЃћрЃарЃћрЃЉрЃА** - рЃДрЃЋрЃћрЃџрЃљ рЃљрЃЦрЃбрЃўрЃБрЃа seller-рЃА рЃЉрЃљрЃќрЃўрЃЊрЃљрЃю
2. **рЃерЃћрЃЦрЃЏрЃюрЃўрЃА/рЃўрЃърЃЮрЃЋрЃюрЃўрЃА рЃЎрЃљрЃбрЃћрЃњрЃЮрЃарЃўрЃћрЃЉрЃА** - рЃЌрЃБ рЃЎрЃљрЃбрЃћрЃњрЃЮрЃарЃўрЃљ рЃљрЃа рЃљрЃарЃАрЃћрЃЉрЃЮрЃЉрЃА, рЃерЃћрЃЦрЃЏрЃюрЃўрЃА
3. **рЃерЃћрЃЦрЃЏрЃюрЃўрЃА 10 рЃАрЃљрЃбрЃћрЃАрЃбрЃЮ рЃърЃарЃЮрЃЊрЃБрЃЦрЃбрЃА**:
   - 2 Jewelry (рЃАрЃљрЃЏрЃЎрЃљрЃБрЃџрЃћрЃЉрЃў)
   - 2 Home Decor (рЃАрЃљрЃ«рЃџрЃўрЃА рЃЊрЃћрЃЎрЃЮрЃарЃљрЃфрЃўрЃљ)
   - 2 Clothing (рЃбрЃљрЃюрЃАрЃљрЃфрЃЏрЃћрЃџрЃў)
   - 2 Accessories (рЃљрЃЦрЃАрЃћрЃАрЃБрЃљрЃарЃћрЃЉрЃў)
   - 2 Art & Crafts (рЃ«рЃћрЃџрЃЮрЃЋрЃюрЃћрЃЉрЃљ рЃЊрЃљ рЃ«рЃћрЃџрЃюрЃљрЃЎрЃћрЃЌрЃЮрЃЉрЃћрЃЉрЃў)

## рЃърЃарЃЮрЃЊрЃБрЃЦрЃбрЃћрЃЉрЃўрЃА рЃЊрЃћрЃбрЃљрЃџрЃћрЃЉрЃў

### рЃЌрЃЋрЃўрЃАрЃћрЃЉрЃћрЃЉрЃў:
- РюЁ **Auto-approved** - рЃДрЃЋрЃћрЃџрЃљ рЃърЃарЃЮрЃЊрЃБрЃЦрЃбрЃў рЃЊрЃљрЃДрЃћрЃюрЃћрЃЉрЃБрЃџрЃўрЃљ `APPROVED` status-рЃўрЃЌ
- РюЁ **Featured products** - рЃърЃўрЃарЃЋрЃћрЃџрЃў 3 рЃърЃарЃЮрЃЊрЃБрЃЦрЃбрЃў featured-рЃљрЃЊ рЃљрЃарЃўрЃА рЃЏрЃЮрЃюрЃўрЃерЃюрЃБрЃџрЃў
- РюЁ **Placeholder images** - рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЉрЃљ Unsplash placeholder images
- РюЁ **Variants** - рЃќрЃЮрЃњрЃўрЃћрЃарЃЌ рЃърЃарЃЮрЃЊрЃБрЃЦрЃбрЃА рЃљрЃЦрЃЋрЃА variants (size, color)
- РюЁ **Complete data** - рЃДрЃЋрЃћрЃџрЃљ рЃЋрЃћрЃџрЃў рЃерЃћрЃЋрЃАрЃћрЃЉрЃБрЃџрЃўрЃљ (material, weight, dimensions, care instructions)

### Seller Distribution:
рЃърЃарЃЮрЃЊрЃБрЃЦрЃбрЃћрЃЉрЃў рЃЌрЃљрЃюрЃљрЃЉрЃарЃљрЃЊ рЃюрЃљрЃгрЃўрЃџрЃЊрЃћрЃЉрЃљ seller-рЃћрЃЉрЃА рЃерЃЮрЃарЃўрЃА.

## рЃерЃћрЃЊрЃћрЃњрЃў

рЃАрЃЎрЃарЃўрЃърЃбрЃўрЃА рЃњрЃљрЃерЃЋрЃћрЃЉрЃўрЃА рЃерЃћрЃЏрЃЊрЃћрЃњ:
- РюЁ 10 рЃърЃарЃЮрЃЊрЃБрЃЦрЃбрЃў рЃерЃћрЃўрЃЦрЃЏрЃюрЃћрЃЉрЃљ Firestore-рЃерЃў
- РюЁ рЃЎрЃљрЃбрЃћрЃњрЃЮрЃарЃўрЃћрЃЉрЃў рЃерЃћрЃўрЃЦрЃЏрЃюрЃћрЃЉрЃљ (рЃЌрЃБ рЃљрЃа рЃљрЃарЃАрЃћрЃЉрЃЮрЃЉрЃА)
- РюЁ рЃърЃарЃЮрЃЊрЃБрЃЦрЃбрЃћрЃЉрЃў рЃЊрЃљрЃЎрЃљрЃЋрЃерЃўрЃарЃћрЃЉрЃБрЃџрЃў рЃўрЃЦрЃюрЃћрЃЉрЃљ seller-рЃћрЃЉрЃЌрЃљрЃю
- РюЁ рЃДрЃЋрЃћрЃџрЃљ рЃърЃарЃЮрЃЊрЃБрЃЦрЃбрЃў рЃЊрЃљрЃДрЃћрЃюрЃћрЃЉрЃБрЃџрЃўрЃљ `APPROVED` status-рЃўрЃЌ

## рЃерЃћрЃфрЃЊрЃЮрЃЏрЃћрЃЉрЃўрЃА рЃЊрЃўрЃљрЃњрЃюрЃЮрЃАрЃбрЃўрЃЎрЃљ

### "No seller users found"
**рЃњрЃљрЃЏрЃЮрЃАрЃљрЃЋрЃљрЃџрЃў**: рЃњрЃљрЃБрЃерЃЋрЃўрЃЌ `npm run seed:users` рЃърЃўрЃарЃЋрЃћрЃџ рЃарЃўрЃњрЃерЃў

### "Firebase credentials are not properly configured"
**рЃњрЃљрЃЏрЃЮрЃАрЃљрЃЋрЃљрЃџрЃў**: рЃерЃћрЃљрЃЏрЃЮрЃгрЃЏрЃћрЃЌ `.env` рЃцрЃљрЃўрЃџрЃў рЃЊрЃљ рЃЊрЃљрЃарЃгрЃЏрЃБрЃюрЃЊрЃўрЃЌ рЃарЃЮрЃЏ Firebase credentials рЃАрЃгрЃЮрЃарЃљрЃЊ рЃљрЃарЃўрЃА рЃЊрЃљрЃДрЃћрЃюрЃћрЃЉрЃБрЃџрЃў

### "PERMISSION_DENIED: Cloud Firestore API has not been used"
**рЃњрЃљрЃЏрЃЮрЃАрЃљрЃЋрЃљрЃџрЃў**: рЃЕрЃљрЃарЃЌрЃћрЃЌ Firestore API Google Cloud Console-рЃерЃў (рЃўрЃ«рЃўрЃџрЃћрЃЌ `FIREBASE_API_SETUP.md`)

## рЃерЃћрЃюрЃўрЃерЃЋрЃюрЃћрЃЉрЃў
